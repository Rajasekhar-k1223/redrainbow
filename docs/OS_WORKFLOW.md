# Multi-OS Security Workflow

## OS Roles
- Kali Linux: general pentesting and recon
- Parrot OS: lightweight pentest and privacy checks
- BlackArch: advanced tools and edge-case testing
- BackBox: standard pentest workflow with clean UI
- Pentoo: live system, wireless and quick hardware tests
- SECHub: web app security testing focus
- CAINE: digital forensics and evidence
- Tails: privacy/anonymity tasks
- Qubes OS: isolated high-risk work
- Security Onion: monitoring and SIEM
- REMnux: malware analysis

## Workflow Stages
1. Recon and mapping: Kali, Parrot, BackBox
2. Web app testing: SECHub
3. Advanced tooling: BlackArch
4. Wireless/live tests: Pentoo
5. Monitoring and detection: Security Onion
6. Forensics and evidence: CAINE
7. Malware analysis: REMnux
8. Privacy tasks: Tails
9. High-risk isolation: Qubes OS
