# RedRainbow Security Platform

This repo is a starter scaffold for a React + FastAPI + MySQL + MongoDB project with a complete, multi-OS cybersecurity program.

## Stack
- Frontend: React
- Backend: FastAPI (Python)
- Databases: MySQL (relational), MongoDB (documents/logs)
- Security lab: Kali, Parrot, BlackArch, BackBox, Pentoo, SECHub, CAINE, Tails, Qubes, Security Onion, REMnux

## Quick Start (high level)
- Read the security plan first: `docs/SECURITY_PLAN.md`
- Review architecture and data mapping: `docs/ARCHITECTURE.md`
- Follow the multi-OS workflow: `docs/OS_WORKFLOW.md`

## Repo Structure
- `backend/` FastAPI service
- `frontend/` React app placeholder
- `docs/` Security and architecture docs
- `infra/` Infrastructure notes and scripts (placeholder)
